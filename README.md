
# Tinper-React 前端开发框架

## 一分钟快速上手

> 需要提前安装全局的 Node 环境

```
$ git clone git@github.com:yonyou-iuap/iuap_pap_react.git
$ cd iuap_pap_react && npm i
$ npm run dev
```
